2. 1주차 주요 개념 키워드 적어보기
버전 관리 : 프로젝트 상태가 변경되는 정보를 관리할 수 있는 툴
Git 초기화 : 컴퓨터에 있는 프로젝트를 git이 관리할 수 있도록 설정
Commit : 현재 프로젝트를 찍어두는 것
Add (Staging) : 커밋할 파일을 선택하는 것
Commit history : 커밋한 순서대로 리스트를 볼 수 있는 곳
Repo : repository의 약자, 원격 repo와 로컬 repo가 있다.
Tracking : 로컬과 원격을 연결하는 것
Push : 로컬의 커밋들을 원격에 밀어넣는 것
Pull : 원격의 커밋들을 로컬로 당겨오는 것
Clone : 원격 repo를 내 컴퓨터로 가져와서 초기 셋팅하는 것
